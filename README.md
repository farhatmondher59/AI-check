# Todo App

A simple Todo App built with HTML, CSS, and JavaScript.

## Features
- Add tasks
- Delete tasks
- Mark tasks as completed
- Responsive design

## How to Use
1. Clone the repo: `git clone <repo-link>`
2. Open `index.html` in your browser
3. Add tasks and manage them easily
